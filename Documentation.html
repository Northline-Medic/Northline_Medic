Project Documentation

This page describes the purpose, methods, data standards, and responsible use of datasets published through Northline_Medic.

Project Mission

Northline_Medic is a student-led GIS project focused on providing educational geospatial resources related to northern Minnesota's native plants, ecosystems, and publicly available environmental information.

The project supports learning, conservation, research, and community engagement through openly shared GIS datasets and mapping resources.

Data Sources

  • Field observations
  • Publicly available government datasets
  • Academic publications
  • Open geospatial data portals
  • Publicly available educational resources

Each dataset includes metadata describing its source, collection date, and any applicable limitations.

Coordinate Reference System

Unless otherwise noted, datasets are published using WGS 84 (EPSG:4326). Alternate coordinate systems may be provided when appropriate for regional analysis.

Responsible Use

Northline_Medic is intended for educational, research, conservation, and community learning purposes. Users are responsible for complying with applicable laws, land ownership, and environmental regulations when using these resources.

Information related to culturally significant plants is presented respectfully and is limited to information that is appropriate for public educational use. Traditional ecological knowledge belongs to Indigenous communities and should be learned through appropriate cultural and community guidance.

Metadata

Each dataset may include:

  • Dataset description
  • Date created
  • Last updated
  • Geographic extent
  • Coordinate reference system
  • Attribute definitions
  • Source information
  • License information
